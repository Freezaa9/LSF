Pour lancer le projet il faut ouvrir LSFDictionary.snl avec Microsoft Visual Studio. 

Ensuite, pour accéder à l'application et l'utiliser, il faut cliquer sur le bouton "play" en haut, bouton de débogage. 
Le projet va alors s'ouvrir sur google chrome ou autre navigateur internet. 

Le code lui est accessible dans l'explorateur de solution avec une hiérarchie bien définie :
les modèles sont par exemple disponible dans le dossier Models.
